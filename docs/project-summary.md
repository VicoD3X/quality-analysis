# Résumé du projet

Quality Analysis transforme un export OpenFoodFacts brut en base nutritionnelle exploitable.

Le projet met en avant :

- la gestion d'un fichier volumineux ;
- l'identification des valeurs manquantes ;
- la sélection de variables nutritionnelles utiles ;
- l'application de règles qualité simples ;
- l'analyse des liens entre nutriments et Nutri-Grade ;
- l'extraction d'une base Python testable.

Le résultat attendu est un projet analytique local, lisible et reproductible, sans surcouche applicative inutile.
