# Dictionnaire de données

## Variables d'identification

- `code` : identifiant produit.
- `product_name` : nom du produit.
- `brands` : marque.
- `countries_fr` : pays de commercialisation.

## Variables nutritionnelles

- `nutrition_grade_fr` : Nutri-Grade de `a` à `e`.
- `nutrition-score-fr_100g` : score nutritionnel France pour 100 g.
- `energy_100g` : énergie pour 100 g.
- `fat_100g` : matières grasses pour 100 g.
- `saturated-fat_100g` : graisses saturées pour 100 g.
- `carbohydrates_100g` : glucides pour 100 g.
- `sugars_100g` : sucres pour 100 g.
- `fiber_100g` : fibres pour 100 g.
- `proteins_100g` : protéines pour 100 g.
- `salt_100g` : sel pour 100 g.
- `sodium_100g` : sodium pour 100 g.
- `additives_n` : nombre d'additifs renseignés.

## Fichiers

- `openfoodfacts-products.tsv` : fichier brut local.
- `openfoodfacts-cleaned.csv` : fichier nettoyé local.
- `openfoodfacts-cleaned-sample.csv` : échantillon versionné.
