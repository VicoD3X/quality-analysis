# Analyses statistiques

## PCA

La PCA est utilisée pour explorer la structure des variables numériques, réduire la dimensionnalité et visualiser les relations entre nutriments.

## ANOVA

L'ANOVA permet de vérifier si les moyennes de certaines variables nutritionnelles diffèrent entre les groupes Nutri-Grade.

Variables étudiées dans le notebook :

- `sugars_100g`
- `saturated-fat_100g`
- `fat_100g`
- `energy_100g`

## CCA

La CCA est utilisée de manière exploratoire pour analyser la relation entre deux groupes de variables nutritionnelles.

## LDA

La LDA sert à visualiser la séparation des classes Nutri-Grade à partir de variables nutritionnelles sélectionnées.

## Limites

Ces analyses restent exploratoires. Les tests statistiques doivent être interprétés avec prudence, notamment lorsque les hypothèses d'homogénéité des variances ne sont pas respectées.
